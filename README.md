# Sheena Ringo – 暗夜の心中立て

> **歌詞登録ステータス：プレーン歌詞のみ**

```
好きと云はれりや誰にでも
体をひらくをんなだと
人がわちきを笑ふのを
知らぬわけではありんせん
だけどこの命
一思ひに投げ出した相手は唯一人だけ
噫こんな奥底を突き止めて置きながら
知らぬ存ぜぬぢや余り然で無いわいな
月に叢雲花に風酔わせておくんなんし
惚れて候

好いたお人にや何処までも
莫迦になる様なをんなだと
人がわちきを笑へども
自棄や勢にじゃありんせん
だからこの心
破いて千切れるまで抱いてよ唯一度だけ
噫こんな痛みなんぞ明日の望みなんぞ
見えぬ聞こえぬ本当も嘘もわいわいな
指切りげんまん針千本飲ませておくんなんし
暗れて候

無けなしの命の証を点しておくんなんし
判るざんせう
```
